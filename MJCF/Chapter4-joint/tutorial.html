第四章：Joint

关节 —— 连接 body、定义运动约束、驱动仿真世界的核心机制

free ball slide hinge stiffness damping range axis
1 关节的本质理解

<joint> 是 MuJoCo 中连接 body 并赋予运动自由度的元件。它的核心作用可以归纳为：

  • 一个 body 内的所有 geom 是一个刚性整体
  • joint 连接当前 body 和父 body，为两者之间引入可动约束
  • 每个 body 最多只能有一个 joint，用于定义当前 body 相对父 body 的运动方式
  • joint 对于上一层 body 是相对静止的；当前 body 围绕 joint 运动

💡 直观理解：body = 骨头，joint = 关节。胳膊（body）内的所有 geom（骨块）是刚性一体的，肘关节（joint）连接大臂（父 body）和前臂（子 body），让前臂能绕肘关节弯曲。

在物理学上：joint 在当前 body 的坐标系中位置固定，父 body 通过 joint 对子 body 施加约束力。

World (root) └── body: torso (躯干) └── joint: free ← 躯干相对世界自由运动 ├── geom: torso_geom ├── body: upper_arm (大臂) │ └── joint: hinge axis=(1,0,0) ← 肩关节：绕X轴转动 │ ├── geom: upper_arm_geom │ └── body: forearm (前臂) │ └── joint: hinge axis=(0,1,0) ← 肘关节：绕Y轴转动 │ └── geom: forearm_geom
2 四种关节类型
F

free

自由度: 6 (3平移+3旋转)

自由关节，赋予body完全自由的运动能力。一般不用，通常用freejoint简写替代。

B

ball

自由度: 3 (仅旋转)

球形关节，绕球心自由旋转。类似人体的肩关节或髋关节。无法平移。

S

slide

自由度: 1 (沿轴平移)

滑轨关节，沿axis方向直线滑动。类似抽屉轨道或气缸活塞。

H

hinge

自由度: 1 (绕轴旋转)

旋转关节（铰链），绕axis旋转。最常用的关节类型，类似门的合页或肘关节。

type自由度运动方式典型应用是否用 axis
free6自由平移+旋转无人机的初始关节、漂浮物体不需要
ball3绕球心三轴旋转肩关节、髋关节、万向节不需要
slide1沿 axis 方向平移活塞、滑轨、抽屉需要
hinge1绕 axis 旋转肘关节、膝关节、门铰链、车轮需要

📌 freejoint 简写：如果想让一个 body 在场景中完全自由运动（如从空中下落），不需要写 <joint type="free"/>，直接写 <freejoint/> 即可（它是 MuJoCo 专门为自由运动提供的便捷简写）。

四种关节用法示例：

🔓 free — 完全自由运动

<body pos="0 0 2">
    <freejoint/>  <!-- 等价于 <joint type="free"/> -->
    <geom type="sphere" size="0.2" rgba=".9 .2 .2 1" />
</body>

⚽ ball — 球形关节

ball 关节让 body 绕关节中心自由旋转但不能平移。ball 不支持 axis 属性。

<body pos="0 0 1">
    <joint type="ball" pos="0 0 0" />
    <geom type="capsule" size="0.05 0.3" rgba=".2 .8 .2 1" />
</body>

🛝 slide — 滑轨关节

slide 关节让 body 沿 axis 方向做直线运动。axis 定义了滑动方向。

<body pos="0 0 1">
    <joint type="slide" pos="0 0 0" axis="0 1 0" range="-0.5 0.5" />
    <geom type="box" size="0.1 0.3 0.1" rgba=".2 .2 .9 1" />
</body>

🔧 hinge — 旋转关节

hinge 关节让 body 绕 axis 旋转。这是构建机器人和机械结构中最常用的关节类型。

<body pos="0 0 1">
    <joint type="hinge" pos="0 0 0" axis="0 0 1" />
    <geom type="box" size="0.05 0.3 0.05" rgba=".9 .7 .2 1" />
</body>
3 关节动力学参数

MuJoCo 的关节不仅定义运动约束，还内置了弹簧-阻尼-惯性模型，可以模拟真实物理关节的动力学特性：

  • pos — 关节相对当前body的位置 (x y z)
  • axis — 活动轴方向 (x y z)，仅 slide 和 hinge 有效
  • stiffness — 弹簧刚度，使关节产生回复力
  • damping — 阻尼系数，抵抗关节速度
  • frictionloss — 关节内部的库仑摩擦力损失
  • armature — 电枢惯性（转子转动惯量 × 减速比² × 传动效率）
  • range — 关节运动范围限制
  • limited — 是否启用范围限制（auto/true/false）
  • ref — 关节角度偏置，即"零位偏移"

3.1 stiffness — 弹簧刚度

使关节像弹簧一样，偏离零位时产生回复力。数值越大，弹簧越硬。

回复力矩 τ = (0 − pos) × stiffness

即：关节偏离参考位置越远，回复力越大。这个力是正比例线性弹簧模型。

3.2 damping — 阻尼系数

产生与速度成正比的阻力，消耗能量，使运动趋于平缓。

阻尼力矩 τ = (0 − v) × damping

damping 让关节的运动速度逐渐衰减，模拟真实的粘性阻尼效果。常用于让关节不要无限振荡。

3.3 frictionloss — 库仑摩擦损失

模拟关节内部的干摩擦（与速度无关，取决于正压力）。不同于 damping 的粘性摩擦，frictionloss 是恒定的摩擦力矩，只要关节在运动就会存在。

3.4 armature — 电枢惯性

表示关节驱动电机转子的等效转动惯量。计算公式：

armature = 转子转动惯量 × 减速比² × 传动效率

在精确的机器人仿真中，电机转子的惯量不能忽略——尤其在高减速比时，armature 项会显著影响关节动力学。通常数值很小（如 0.001）。

💡 参数选择建议：

  • 弹簧关节：设 stiffness > 0，damping > 0（防止振荡）
  • 柔顺关节：小 stiffness + 适中 damping
  • 硬关节（位置控制）：stiffness=0，damping 根据需要设置
  • 真实电机仿真：加上 armature 值
4 range — 关节运动范围限制

range 定义了关节的运动行程限制，配合 limited 属性使用：

  • hinge / slide：range 为两个值 "min max"，分别是最小和最大角度/位移
  • ball：只有两个参数有效，第一个设为 0，第二个为锥角限制
  • limited="auto"：自动根据 range 的数值判断是否启用限制。如果 range 的两端都为 0，则视为无限制
  • 需要在 <compiler> 中设置 autolimits="true"
<!-- hinge关节，限制旋转角度在 -30° 到 60° 之间 (弧度) -->
<joint type="hinge" pos="0 0 0" axis="0 0 1"
       range="-0.523 1.047" limited="true" />

<!-- slide关节，限制滑动范围 -0.3m 到 0.3m -->
<joint type="slide" axis="0 1 0"
       range="-0.3 0.3" limited="true" />

<!-- ball关节：锥角限制 45° (0.785 rad) -->
<joint type="ball" range="0 0.785" limited="true" />

⚠️ 注意：在 compiler 中设置 autolimits="true" 后，MuJoCo 会根据 range 值自动推断 limited 属性，无需手动指定。

5 axis — 关节活动轴

axis 定义了 hinge 和 slide 关节的运动方向。它是一个 3D 向量 (x y z)，在父 body 的坐标系中定义。

  • hinge：axis 是旋转轴，关节围绕此轴旋转（右手定则）
  • slide：axis 是滑动方向，关节沿此轴平移
  • axis 向量会自动归一化，所以 "1 0 0" 和 "2 0 0" 效果相同
axis 方向与运动的关系： axis="1 0 0" → hinge绕X轴旋转 / slide沿X轴滑动 axis="0 1 0" → hinge绕Y轴旋转 / slide沿Y轴滑动 axis="0 0 1" → hinge绕Z轴旋转 / slide沿Z轴滑动 axis="0 1 1" → hinge绕(0,1,1)方向旋转（对角的合页）
6 ref — 角度偏置

ref 属性定义了关节的"零位"参考角度。它与 stiffness 配合使用时非常重要：

  • 关节的当前角度 = qpos − ref
  • 弹簧回复力是基于 (ref − 当前角度) 计算的
  • 相当于你可以定义"这个关节的正常位置是 30°"，弹簧会试图把关节保持在 30°
<!-- 关节的默认位置偏置为0.3rad(约17°)，弹簧会把关节推向这个位置 -->
<joint type="hinge" axis="0 0 1"
       ref="0.3" stiffness="0.5" />
7 完整场景解析

scence.xml 展示了一个精巧的双级摆系统（倒立摆变体），包含了两个 hinge 关节的级联：

World └── body: support pos=(0,0,0.1) ├── geom: cylinder (支撑柱) └── body: rotary_arm pos=(0,0,0.51) └── joint: hinge "pivot" axis=(0,0,1) 阻尼0.001 刚度0.5 ├── geom: capsule (水平旋转臂) └── body: pendulum pos=(0.2,0,0) └── joint: hinge "ph" axis=(1,0,0) 阻尼0.001 ├── geom: capsule (摆杆) └── geom: sphere (配重球)

关节1: pivot (hinge, axis=0 0 1)

水平旋转关节，让 rotary_arm 绕 Z 轴在水平面内旋转。stiffness=0.5 提供了适中的弹簧回复力。

<joint type="hinge" name="pivot" pos="0 0 0"
       axis="0 0 1" damping="0.001"
       frictionloss="0.0" stiffness="0.5" />

关节2: ph (hinge, axis=1 0 0)

垂直旋转关节，让 pendulum 绕 X 轴在垂直面内摆动。damping=0.001 提供轻微阻尼防止无限摆动。

<joint type="hinge" name="ph" pos="0 0 0"
       axis="1 0 0" damping="0.001"
       frictionloss="0.0" />

📌 关键观察：

  • pivot 关节有 stiffness=0.5，所以水平臂偏离零位后会自动弹回
  • ph 关节没有 stiffness，所以摆杆在垂直面内自由摆动
  • 配重球（sphere）放在摆杆末端，增大惯性，让摆动更明显
8 本章小结
概念核心要点
joint 本质连接父子 body，定义相对运动约束；每 body 最多一个 joint
free / freejoint6 自由度完全自由运动；飞行物体、根 body 常用
ball3 旋转自由度，无平移；模拟万向节/肩关节
slide1 平移自由度，沿 axis 直线运动；需指定 axis
hinge1 旋转自由度，绕 axis 旋转；最常用关节，需指定 axis
stiffness弹簧回复力 τ = (0−pos)×stiffness；数值越大越硬
damping粘性阻尼 τ = (0−v)×damping；消耗能量、平缓运动
frictionloss恒定摩擦力矩（库仑摩擦），与速度无关
armature电机转子等效惯量，高减速比时影响显著
range & limited关节行程限制；hinge/slide 二值；ball 锥角；配合 autolimits 使用
ref角度偏置，弹簧试图将关节保持在此位置
